### 择校
一战的时候什么信息都没看，就报了中南，听信了同学22ds很好考的建议，然后自己摆烂下岸。二战在北师和华工之间选择了北师，因为亲戚推荐我去北京发展，然后408考了86，喜提被京区985卡90分的408单科线。三战考虑了科大先研，南大智科，先研是24华五初试分数最低的院，南大智科25新改408，学费低有宿舍，就是复试科目比较多。报名的时候，考虑再三，还是顶不住压力，选择了11408最大池子哈工，冲深圳保卓工。择校尽量还是不要随大流，报浙软，南软这种大流量的，事实也证明这些学校很炸裂，大部分的人都在第一层，我们只需要在第二层就行了，比如科大智院从365降到300爆冷。哈工实际上是小炸的，但是由于扩招200人，从330的线降到了310，赢麻了。选哈工除了名额多，还有一个原因，就是在[csrankings](https://csrankings.org/#/fromyear/2018/toyear/2024/index?all&cn)上，哈工进了AI前十，和南大人大位置差不多，选大导也可以在csrankings上找。更详细择校过程可以参考[谈谈26考研11408如何择校](https://zhuanlan.zhihu.com/p/1895969938834949741)

### 初试
|科目|学习方法|
|--|--|
|政治|看徐涛老师的马哲部分网课，肖秀荣背诵手册多看几遍，使用苍盾小程序刷题，刷肖1000题以及模拟卷选择题|
|英语|使用欧路词典背单词，反复做真题|
|数学|看同济高数教材和浙大概率论教材，写李林880或张宇1000题，刷真题和张宇、李艳芳、欧几里得等模拟卷|
|408|把王道书多看几遍，多刷几遍王道题、真题，刷软微佬编的408的1000题，可以加浙大软微考研群，群友回答问题比较积极| 

### 复试
初复试5：5，机试1分等于初试1.875分，务必认真对待。面试准备AI相关的知识。本科经历三无，所以面试低也没办法，只能卷机试了。

|科目|学习方法|
|--|--|
|数据库|做哈工大数据库MOOC题目，要做到熟练背答案的程度|
|离散数学|看屈书（要补充连续统和理想方面的知识点），然后做哈工大集合论和图论MOOC，和近世代数网易云题目，看复习PPT|
|算法|做哈工大算法MOOC题目，看复习PPT|
|C语言|刷[SSE](https://sse.hit.edu.cn/t/#/login)中等和困难，特别注意结构体，排序，指针，字符串的题目| 
|逻辑题|练练说真话差不多了| 
|人工智能|看刘二大人的[PyTorch入门教程](https://www.bilibili.com/video/BV1Y7411d7Ys)和[StatQuest](https://www.bilibili.com/video/BV1Km421u7uu)的NLP科普|

### 心路历程
二战北师失败了，不想调剂，经历很多糟心的事，开始怀疑自己考研的初心，但是我觉得自己还是没有尽力，所以在25考研继续努力了一年，学习过程并不一帆风顺，408总是没有正反馈，掌握的效果不好，只能堆时间，做1000题，各机构模拟题，考场上非常粗心，最后只能拿到105，25英语是跌的最惨的，比24英语低了10多分，我这年还努力准备了作文，跟的陈曲老师，但是阅读炸了。数学刷了很多题，但是可能是考试前几天没做试卷，考试的时候，脑子转的很慢，大题很卡，好在最后能做出来的大题都做出来了，线面积分，概率论第二题放了，证明题第二问也扣掉了，选填由于粗心错了两个。政治是不断刷模拟卷，背肖四，非常难背，就按葫芦背书法一点一点背，背到后面政治话术都知道编了，黑龙江水区，所以最后结果还不错，选择题有40分。

25初试结束后，就尽全力准备机试内容，对过答案知道自己差不多350，初试分没有优势，整个过程非常紧张，逼自己像初试一样学习，看数据库看的想困，还是坚持下来了，最后出复试名单的时候，才发现分数线310，历史最低，抄到今年最大的底，放心了点，至少能稳进大名单了。

3月25号飞到哈尔滨，又在哈尔滨的自习室学了3天，就去考试了，最后发现机试也过度准备了，很多学的没考，还是靠直觉蒙了很多题，好在结果是好的，不过C语言绝对不能丢分，丢分一道题就相当于初试被甩18分。面试自我感觉良好，做到了不冷场，坦诚和自信，但是由于三无，拿的分也低。

回看25考研，选哈工非常正确，本来是抱着考380的心态选的哈工深，最后只有350，但惊险上岸了，选先研或南大智科，真的要出事。哈工真的名额多，双盲，机试占比高，真的就是纯看实力，每一步都需要精心准备，过程惊心动魄。考了3次，失败了2次，每一次都是对自信心的摧毁，反复陷入习得性无助，只能靠自己调整心态顶住，最后考完都没有什么开心的感觉，因为不是高分上岸，每一步准备的都很辛苦，结束了都不知道怎么放松，也不敢太放松，很快就会进组了，接着就是紧凑的研究生学习了。我会延续努力学习的作风，凡事都尽力准备，希望都能有好结果吧。

考研有非常多的信息差，择校，找初试复试资料，选导师等等的，所以一定要学会搜索信息。
